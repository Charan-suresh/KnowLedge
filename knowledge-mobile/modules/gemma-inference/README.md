# Gemma Inference Native Module

This folder is the Expo module scaffold for native Gemma 4 inference.

Android target:
- ML Kit GenAI Prompt API
- Streaming token events
- Multimodal image/audio input

iOS target:
- MediaPipe LLM Inference API
- Streaming token events
- Multimodal image/audio input

The current files are contract stubs so the Expo app can compile and the native bridge can be implemented in a prebuild or bare workflow.
