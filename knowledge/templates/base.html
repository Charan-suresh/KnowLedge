How KnowLedge Works

The Problem

Every student deserves a Socratic tutor — not just the privileged few.

Research shows that Socratic questioning — being pushed to explain your reasoning rather than just recall facts — produces durable understanding. But access to that kind of rigorous one-on-one tutoring is expensive and scarce. KnowLedge changes that using Gemma 4.

🤗 unsloth/gemma-4-4b-it 📖 100% Open Source

The Four Agents

Scout

Concept Extractor

Paste any study content — textbook excerpts, lecture notes, your own summaries. Scout sends it to Gemma 4 and extracts 3–8 key concepts you need to understand.

Sage

Socratic Tutor

Click "Sage →" on any concept. Gemma 4 takes on the role of a Socratic tutor — it never gives you the answer directly. It asks escalating questions until your explanations demonstrate genuine mastery.

Lens

Vision Verifier

Photo of handwritten notes? Drop it in Lens. Gemma 4 vision analyses whether your reasoning is genuine, identifies which tracked concepts are present, and flags gaps or misconceptions.

Solo Mode

Timed Recall Test

The gold standard of durable learning. Click "Solo" on any concept and explain it from memory without any prompting. Gemma 4 scores your answer strictly.

Integrity System

Hard to game by design.

Every Sage session generates a signed cryptographic fingerprint tracking typing rhythm, paste events, response timing, and turn-by-turn entropy. If a session looks suspicious it gets flagged as "Persists" and redirected to Solo Mode.

Local Setup

Run everything locally.

  1. Clone: git clone https://github.com/Charan-suresh/KnowLedge.git
  2. Install: pip install -r requirements.txt
  3. Pull model: ollama pull gemma4:e4b
  4. Start Ollama: ollama serve
  5. Start app: uvicorn knowledge.main:app --reload
