One model, four roles

KnowLedge now runs entirely on local Ollama.

Scout, Sage, Lens, and Solo all use the same Gemma model. Change only the Ollama URL in the config modal. There are no per-role model settings anymore.

  1. Install Ollama.
  2. Run ollama pull gemma4:e4b-q4_K_M.
  3. Run ollama serve.
  4. Open the Ollama config modal and click Detect →.
