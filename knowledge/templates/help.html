The Problem

Every student deserves a Socratic tutor — not just the privileged few.

Research shows that Socratic questioning — being pushed to explain your reasoning rather than just recall facts — produces durable understanding. But access to that kind of rigorous one-on-one tutoring is expensive and scarce. KnowLedge changes that using Gemma 4.

🏆 Kaggle Gemma 4 Good Hackathon 🤗 unsloth/gemma-4-4b-it 📖 100% Open Source

Scout

Concept Extractor

Paste any study content — textbook excerpts, lecture notes, your own summaries. Scout sends it to Gemma 4 and extracts 3–8 key concepts you need to understand, adding each to your personal knowledge debt ledger.

Sage

Socratic Tutor

Click "Sage →" on any concept. Gemma 4 takes on the role of a Socratic tutor — it never gives you the answer directly. It asks escalating questions until your explanations demonstrate genuine mastery, then marks the concept as cleared.

Lens

Vision Verifier

Photo of handwritten notes? Drop it in Lens. Gemma 4 vision analyses whether your reasoning is genuine, identifies which tracked concepts are present, and flags gaps or misconceptions — all from a single image.

Solo Mode

Timed Recall Test

The gold standard of durable learning. Click "Solo" on any concept and explain it from memory without any prompting. Gemma 4 scores your answer strictly but fairly — partial credit isn't enough to mark a concept cleared.

Integrity System

Hard to game by design.

Every Sage session generates a signed cryptographic fingerprint tracking typing rhythm, paste events, response timing, and turn-by-turn entropy. If a session looks suspicious — e.g. copy-pasted textbook answers — it gets flagged as "Persists" and redirected to Solo Mode for independent verification.

Anti-gaming probes interrupt conversations at random intervals, demanding specificity on the exact phrase a student just typed. This makes paste-and-run attacks ineffective.

Local Setup

Run everything on your own machine.

  1. Clone: git clone https://github.com/Charan-suresh/KnowLedge.git && cd KnowLedge
  2. Install: pip install -r requirements.txt
  3. Pull model: ollama pull gemma4:e4b
  4. Start Ollama: ollama serve
  5. Start app: uvicorn knowledge.main:app --reload
  6. Open: localhost:8000

No internet required after setup. No API keys. No data leaves your machine. The entire inference stack runs on Gemma 4 via Ollama.

Architecture

Technical Stack

Backend: FastAPI (Python) · SQLite (local ledger) · ChromaDB (RAG vector store) · Gunicorn + Uvicorn workers on Render

AI: Gemma 4 (unsloth/gemma-4-4b-it) via either local Ollama or a Hugging Face Space FastAPI backend. The inference router transparently switches between backends based on the INFERENCE_BACKEND env variable.

Integrity: HMAC-SHA256 session fingerprinting · Paste detection · Temporal entropy scoring · Anti-gaming Socratic interrupt probes

Frontend: Jinja2 server-side templates · Vanilla JS (no framework) · CSS custom properties with glassmorphism design system
